# Dicee - The Dice Game
A fun two-player dice game where the dice roll automatically every time you refresh the page!

##  How to Play
- Open the page and the dice will roll automatically
- The player with the higher number wins
- Refresh the page to roll again!

##  Features
- Random dice roll on every page refresh
- Automatically displays the winner
- Clean and responsive UI

##  Built With
- HTML
- CSS
- JavaScript

##  Preview
Player 1 and Player 2 each get a random dice (1-6).
The higher number wins. If both are equal — nobody wins!
